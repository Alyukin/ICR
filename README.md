# ICR
Интеллектуальное распознавание рукописного кириллического текста (ICR).
В отличие от стандартных OCR-систем, данный проект нацелен на работу с неструктурированным рукописным вводом, учитывая особенности связного русского письма и индивидуальных почерков.

### Технологический стек
* **Core:** `Python`, `C++`
* **Environment:** `Linux`
* **Focus:** Computer Vision, Deep Learning, Image Preprocessing
